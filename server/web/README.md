# WEB 端服务

这个是一个后端接口服务，同时内嵌静态文件服务。

## 编译与调试

### 调试模式

```shell
cargo run
```

### 生产模式

```shell
cargo build
```

## 服务

- [前端服务](http://127.0.0.1:8000/)
- [后端服务](http://127.0.0.1:8000/api/v1/)
- [swagger-ui](http://127.0.0.1:8000/swagger-ui/)
