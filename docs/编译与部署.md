# 编译与部署

## 后端

### App/Web

- 项目路径: server/app/web

- 开发模式

```shell
cargo run --package web
```

- 发布模式

```shell
cargo build -r --package web
```

## 前端

## 前端更新所有依赖

更新所有依赖项，此操作会忽略 package.json 指定的范围。

参考文档: https://www.bookstack.cn/read/pnpm-8-zh/b665a0f2f935aafe.md

### 更新所有依赖到当前兼容的最新版本

注意: 可能存在依赖不兼容的风险。

```shell
pnpm up
```

### 更新所有依赖到最新

注意: 可能存在依赖不兼容的风险。

```shell
pnpm up —latest
```
